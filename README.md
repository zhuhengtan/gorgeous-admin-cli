# gorgeous-admin介绍

### 前端

使用react+ts+antd搭建的管理端，此cli旨在快速初始化gorgeous-admin管理端项目

### 后端

使用golang+gin框架开发


# 使用

1. 全局安装cli
```
sudo npm i gorgeous-admin-cli -g
```

2. 在工作目录初始化项目即可
```
gorgeous-admin-cli create my-admin
```
